Présentation¶

Gesta est un logiciel de gestion des participants au seins d’une Eft/Asbl, elle prend en compte la gestion du stagiaire durant son processus de formation.

Gesta permet entre autres de gérer :

  • Une inscription.
  • Une gestion de contrat.
  • Les prestations de présences.
  • De calculer les statistiques.
  • De générer les documents demandés par les organismes subsidiant
